# CatPlatformer

A small platformer prototype built with the **Godot game engine**.

This project was an early experiment with platformer mechanics, including player movement, jump pads, portals between levels, and basic environment interactions.

The game is unfinished and was mainly used for learning and experimentation.

---

## Features

- Basic 2D platformer movement
- Custom jump pads
- Portals between levels
- Simple interaction mechanics
- Multiple prototype scenes

---

## How to Run

1. Install the **Godot Engine** (version 3.x or 4.x depending on compatibility).
2. Open Godot.
3. Click **Import Project**.
4. Select the `project.godot` file in this repository.

The project will then load inside Godot and can be run from the editor.

---

## Project Structure

```
assets/   - sprites and game assets
scenes/   - game levels and scene files
scripts/  - player logic and game mechanics
```
